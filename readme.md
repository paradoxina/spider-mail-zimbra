# Spider Mail

To download emails to zimbra mail.

# Install

install node js 

install dependency

```
npm install 
```

## Usage

Normal:

fill in the fields in config.json

```
{
  "users": [
    {
      "login": "",
      "password": ""
    },
    {
      "login": "",
      "password": ""
    }
  ],
  "url": ""
}
```

### Run

```
npm start
```
